CheatSheet - Selector

Selector Description Example
* this element targets all elements * { margin: 0; }
.class This will select all elements with the named class. h1 class= "sheet"
#id #id="rogue" This will select elements with id=rogue.
Chaining defines and group elements to which css rule set applies. h1, h2 color: red;}
specificity Order by which the browser decides which css will be displayed. h1#header {color: red;} .headline{ color:firebrick;} id wins.
attribute The attribute selector can be used to target elements containing attributes. [href] { color: magenta;}
pseudo-class selector Elements can change based on user interactions. p:hover {background-color: red;} paragraph changes to red.
:focus Another type of pseudo selector. When you click an input, a border is added showing it is in focus.
:visited The link changes color to show links you have visited. a:visited {color:green;}>/td>
:hover Another type of pseudo selector. a :hover { } Shows mouse hover over an element.
:visited The link changes color to show links you have visited. a:visited {color:green;}
p This type selector target the paragraph. p Will change the appearance of the paragraph
